# GymBuddy 💪

A personalised AI workout plan generator built for beginners.

## What it does
- User inputs their goal, available equipment, fitness level and intensity
- AI generates a custom workout plan with sets, reps and muscle groups
- YouTube tutorial link for every exercise

## Setup
1. Clone the repo
2. Get a free Gemini API key at https://aistudio.google.com
3. Open `index.html` in VSCode with Live Server
4. Add your API key in `script.js`

## Stack
- HTML, CSS, vanilla JS
- Gemini API (google-generative-ai)
